# Checklist Qualite Call VERIDIS

Objectif : auto-evaluer chaque call en 3 minutes pour ameliorer la conversion sans devenir agressif.

Score : 0 = non, 1 = partiel, 2 = oui.

## 1. Preparation

| Controle | Score |
| --- | --- |
| J'ai verifie le role, l'entreprise, le secteur et l'effectif |  |
| J'ai identifie une hypothese de douleur avant le call |  |
| J'ai ouvert la fiche Notion avant le call |  |
| J'avais un exemple ou rapport proche du secteur |  |
| Je connaissais le plan probablement pertinent |  |

## Interpretation

- 80-100 : call solide, analyser surtout les objections.
- 60-79 : bon call, ameliorer qualification ou next step.
- 40-59 : call trop flou, revoir decouverte et impact.
- 0-39 : call non scalable, reprendre le script minute par minute.

## Commentaire apres checklist

La checklist n'est pas la pour se juger. Elle sert a rendre la vente reproductible, sans transformer le fondateur en vendeur agressif.
