# CSRD Scope 3 : Le Guide Complet pour les Entreprises Francaises 2026

Le Scope 3 est souvent la partie la plus inconfortable d'un reporting climat. Pas parce que le concept est impossible a comprendre. Parce qu'il oblige l'entreprise a regarder au-dela de ses murs : fournisseurs, transport, achats, deplacements, usage des produits, fin de vie.

En 10 minutes, vous allez comprendre ce qu'est le Scope 3, pourquoi il compte dans ESRS E1, les erreurs qui fragilisent un rapport et comment lancer un diagnostic propre sans attendre d'avoir des donnees parfaites.

## Qu'est-ce que le Scope 3 en 3 phrases ?

Vos emissions de gaz a effet de serre se divisent en trois familles.

| Scope | Definition simple | Exemples | Niveau de controle |
| --- | --- | --- | --- |
| Scope 1 | Emissions directes de l'entreprise | Chaudieres, vehicules detenus, procedes industriels | Fort |
| Scope 2 | Emissions liees a l'energie achetee | Electricite, chaleur, vapeur, froid achetes | Moyen |
| Scope 3 | Autres emissions indirectes de la chaine de valeur | Achats, transport, deplacements, usage des produits, fin de vie | Faible a moyen |

Le Scope 3, c'est donc l'empreinte carbone cachee dans votre ecosysteme. Il est moins facile a mesurer, mais souvent essentiel pour comprendre votre exposition reelle.

Analogie simple : si votre entreprise etait un restaurant, le Scope 1 serait le gaz utilise en cuisine, le Scope 2 l'electricite du local, et le Scope 3 les ingredients, les livraisons, les emballages, les trajets clients et les dechets.

## Pourquoi le Scope 3 compte dans la CSRD ?

Le standard ESRS E1 couvre le changement climatique. Dans ce standard, l'exigence E1-6 porte sur les emissions brutes de gaz a effet de serre, y compris les Scopes 1, 2 et 3 lorsque l'information est materielle.

Dit autrement : si le Scope 3 est significatif pour votre modele economique, votre chaine de valeur ou vos impacts climatiques, vous devez etre capable d'expliquer comment vous l'avez traite.

Le point important pour un DAF : la CSRD ne demande pas seulement un chiffre. Elle demande une information exploitable, tracee et explicable :

- quelles categories sont incluses ;
- quelles categories sont exclues et pourquoi ;
- quelle methode de calcul est utilisee ;
- quelles donnees sont mesurees, estimees ou extrapolees ;
- quelles limites et incertitudes restent ouvertes ;
- comment la qualite des donnees va s'ameliorer.

### Calendrier : prudence obligatoire

Le calendrier CSRD a ete modifie au niveau europeen par la directive dite "stop-the-clock". Avant de publier une date precise dans un contenu commercial, verifiez toujours les sources officielles du jour.

La bonne posture : ne basez pas votre chantier Scope 3 uniquement sur une date. Basez-le sur la capacite a prouver votre methode, car vos clients, banques, investisseurs et donneurs d'ordre peuvent demander ces donnees avant meme une obligation formelle.

## Les 5 erreurs Scope 3 les plus frequentes

### 1. Dire "nous n'avons pas de Scope 3"

Meme une entreprise de services a du Scope 3 : achats informatiques, cloud, bureaux, deplacements, prestations externes, evenements, dechets.

Consequence : le rapport donne l'impression que le sujet a ete ignore.

Solution rapide : partez des 15 categories du GHG Protocol et marquez chaque categorie comme "applicable", "non applicable" ou "a verifier".

### 2. Ne pas justifier les categories exclues

Exclure une categorie peut etre legitime. Ne pas l'expliquer est fragile.

Consequence : l'auditeur, le client ou le comite de direction ne sait pas si l'exclusion est une decision methodologique ou un oubli.

Solution rapide : ajoutez une colonne "justification" dans votre mapping Scope 3. Une phrase claire vaut mieux qu'un silence.

### 3. Chercher la donnee parfaite avant de commencer

Le Scope 3 commence souvent avec des estimations. C'est normal. Le risque n'est pas d'estimer ; le risque est d'estimer sans methode documentee.

Consequence : l'equipe attend, le calendrier glisse, puis tout se fait sous pression.

Solution rapide : classez les donnees en trois niveaux : mesure directe, donnee fournisseur, estimation sectorielle. Puis construisez un plan d'amelioration.

### 4. Melanger depenses et volumes sans logique

Une approche "spend-based" peut servir au demarrage, mais elle doit etre clairement documentee. Une approche "activity-based" peut etre plus precise, mais demande plus de donnees.

Consequence : le chiffre Scope 3 devient difficile a defendre.

Solution rapide : indiquez pour chaque categorie la methode utilisee, la source du facteur d'emission et la limite principale.

### 5. Ne pas transformer le calcul en plan d'action

Un inventaire Scope 3 qui finit dans un PDF n'aide pas beaucoup le business.

Consequence : le rapport existe, mais il ne pilote rien.

Solution rapide : associez les plus gros postes a une action : fournisseur a interroger, politique achat a revoir, donnees transport a fiabiliser, cible a definir.

## Comment faire votre Scope 3 etape par etape

### Etape 1 : cartographier les categories

Prenez les 15 categories du GHG Protocol et faites un premier tri.

| Priorite | Question | Exemple |
| --- | --- | --- |
| Haute | La categorie pese-t-elle probablement lourd ? | Achats de matieres, transport aval |
| Moyenne | La categorie est-elle demandee par les clients ? | Deplacements, usage produit |
| Basse | La categorie semble-t-elle non applicable ? | Franchise si aucune franchise |

Objectif : ne pas calculer tout tout de suite. Objectif : identifier ce qui merite une preuve.

### Etape 2 : partir des donnees deja disponibles

Commencez par les sources internes :

- grand livre achats ;
- base fournisseurs ;
- notes de frais ;
- donnees transport ;
- consommation energetique ;
- donnees RH ;
- ERP et outils achats.

Ne cherchez pas encore la perfection. Cherchez la tracabilite.

### Etape 3 : choisir une methode par categorie

Trois approches reviennent souvent :

- spend-based : estimation a partir des montants depenses ;
- activity-based : calcul a partir de volumes physiques ;
- supplier-specific : donnees fournies par les fournisseurs.

La bonne methode depend du niveau de maturite. Pour un premier diagnostic, l'important est d'indiquer la methode, ses limites et le plan d'amelioration.

### Etape 4 : documenter les hypotheses

Chaque hypothese doit etre lisible par quelqu'un qui n'a pas fait le calcul.

Document minimal :

- source de la donnee ;
- periode couverte ;
- facteur d'emission ;
- methode ;
- responsable ;
- limite connue ;
- prochaine amelioration.

### Etape 5 : prioriser les corrections

Tous les gaps ne se valent pas.

Priorisez selon quatre criteres :

1. poids probable dans les emissions ;
2. risque de non-conformite ;
3. difficulte de collecte ;
4. importance pour les clients ou investisseurs.

### Etape 6 : preparer la preuve

Posez cette question : "Si on nous demande la preuve demain, que montrons-nous ?"

Si la reponse est floue, le gap est reel.

## Questions frequentes

### Mon entreprise n'est pas encore directement concernee par la CSRD. Dois-je travailler le Scope 3 ?

Oui si vous vendez a des clients concernes, si vous repondez a des appels d'offres ou si vos financeurs demandent des donnees climat. La pression peut venir de la chaine de valeur avant l'obligation.

### Puis-je utiliser des estimations ?

Oui, si vous les documentez proprement. Le point cle est de distinguer ce qui est mesure, estime ou fourni par un tiers.

### Dois-je couvrir les 15 categories ?

Vous devez les passer en revue. Toutes ne seront pas forcement materielles ou applicables, mais les exclusions doivent etre justifiees.

## Conclusion

Le Scope 3 n'est pas un exercice de perfection. C'est un exercice de transparence progressive.

Retenez trois choses :

1. le Scope 3 regarde votre chaine de valeur, pas seulement vos murs ;
2. une estimation documentee est plus utile qu'un silence ;
3. le vrai sujet est la preuve : methode, source, limite, amelioration.
