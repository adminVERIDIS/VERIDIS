# Checklist CSRD 2026

## Page 1 - Promesse

La CSRD peut sembler enorme. Cette checklist sert a faire une chose simple : identifier rapidement ce qui est pret, partiel ou absent dans votre dossier.

Utilisez-la avant :

- un comite finance/RSE ;
- un atelier de double materialite ;
- une mission consultant ;
- une revue interne ;
- un upload dans VERIDIS.

Regle d'usage : ne cochez une case que si vous avez une preuve.

Score rapide :

- 0 point : absent ;
- 1 point : partiel ;
- 2 points : pret avec preuve.

CTA : pour obtenir un score automatique, chargez votre rapport dans VERIDIS.
